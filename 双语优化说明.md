## 🎯 双语格式规范

### 标准格式

采用统一的双语格式：**English(中文)**

```
✅ 正确示例：
- Company Name (公司名称)
- Service Fee (服务费)
- Required (必须)
- 10 days (天)

❌ 错误示例：
- 公司名称(Company Name)  // 顺序错误
- Service Fee（服务费）   // 括号类型错误
- Required必须            // 缺少括号
```

### 特殊情况处理

#### 1. 带数值的文本
```
格式：数字 + English(中文)
示例：10 days (天)
示例：3 professional service providers (家专业服务商)
```

#### 2. 比例/百分比
```
格式：Percentage(中文) (%)
示例：Shareholding Percentage (持股比例) (%)
示例：Tax Rate (税率): 8.84%
```

#### 3. 日期时间
```
格式：English(中文)
示例：Working Days (工作日)
示例：Calendar Days (自然日)
```

#### 4. 列表项目
```
格式：English(中文)
示例：Required (必须)
示例：Optional (可选)
示例：Yes (需要) / No (不需要)
```
